# Mint Investment Strategies Front-End

Premium, dark-mode, multi-page React + TypeScript + Vite marketing site for Mint Investment Strategies.

## Stack
- React + TypeScript + Vite
- TailwindCSS
- Framer Motion
- React Router

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy (Render static site)
- Build command: `npm run build`
- Publish directory: `dist`
